1-Installare Eclipse
2-importare il progetto "FakeSocialTest" su eclipse
3-importare Javax.json 1.0 nel buildpath del progetto (presente in questa root)
4-Creare un server tomcat 7.0 ed aggiungervi "FakeSocialTest"
5-Aprire: "......\workspace\FakeSocialTest\src\myServlets\MyServlet.java" e modificare la variabile
"percorso", inserendo il percorso completo legato al proprio computer, riferito al file:
"......../FakeSocialTest/WebContent/templates/users.txt"
6-Aprire un browser e inserire l'url: "localhost:8080/FakeSocialTest/MyServlet"
